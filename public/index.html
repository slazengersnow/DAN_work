実装方法の説明

1. HTMLにCSSを追加する

<style>
/* 年度・月表示非表示化用のCSS */
.yearMonthDisplay,
.yearMonthSection,
.filterSection,
/* 他のセレクタ... */
{
  display: none !important;
  visibility: hidden !important;
  height: 0 !important;
  overflow: hidden !important;
}

/* タブは確実に表示するためのCSS */
.tabsContainer,
.monthly-tab,
/* 他のタブセレクタ... */
{
  display: block !important;
  visibility: visible !important;
  opacity: 1 !important;
}
</style>

2. JavaScriptスクリプトを追加する

<script>
// 月次報告画面の年度・月表示を強制的に非表示にするスクリプト
(function() {
  console.log('年度・月表示の非表示化スクリプトを実行開始');
  
  // メイン処理...
  function forceHideYearMonth() {
    // 年度・月表示を非表示にする処理
    // タブは表示を維持する処理
  }
  
  // 実行...
  forceHideYearMonth();
  
  // MutationObserverでDOM変更を監視...
})();
</script>

動作確認

このデモページでは、CSSにより年度・月選択部分が非表示になっています。

実際のアプリケーションでは、より複雑なDOM構造に対応するため、JavaScriptによる検出と非表示化が重要です。

詳細な実装については、「year-month-hider-generator.js」スクリプトによって生成される各ファイルを参照してください。
